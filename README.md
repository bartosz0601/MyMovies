# MyMovies
Vue.js + ASP.NET Core

## General info
Web application for managing movies. 

## Setup
To run API project(server):
```
$ cd ..\mymovies\api
$ dotnet run
```
To run clientapp(webapp):
```
$ cd ..\mymovies\clientapp
$ npm install
$ npm run dev
```
Run on your browser: http://localhost:5173/

## Technologies
Project is created with:
* ASP.NET Core 6.0
* EntityFramework
* MediatR
* AutoMapper
* Vue.js 3.2.45
* Bootstrap 5
* Axios
* Vuelidate
